Search Engine
-------------
	
This is a tiny search engine uses 3 leves of cache
	
 1. in memory
 2. historic seach
 3. lucene search	

Technologies
----

 - Java 8
 - Spring Boot 1.4.0
 - JPA 2.1
 - Hibernate 5.0.9
 - Hibernate Search 5.5.4
 - Lucene 5.3.1
 - HSQL 2.3.3

REST Endpoints
--------------

 2. save
 3. delete
 4. clear
 5. getall
